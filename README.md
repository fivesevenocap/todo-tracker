# Todo Tracker (Python)

Простое консольное приложение для управления задачами.

Проект написан на Python и позволяет добавлять, редактировать и отслеживать задачи через терминал.

## Возможности

- добавление задач
- удаление задач
- переименование задач
- отметка задачи как выполненной
- просмотр выполненных задач
- поиск задач
- статистика задач
- сохранение данных в JSON

## Структура проекта
```
todo_tracker
│
├── main.py
├── tasks.py
├── utils.py
├── storage.py
├── tasks.json
└── README.md
```

## Запуск проекта
Запустить программу можно командой:
```bash
python3 main.py
```
